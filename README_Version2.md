# Ship Draft Survey Autocalc (FastAPI MVP)

## Описание

Веб-приложение для автоматизированного draft survey.  
Встроенный demo-профиль (демонстрационные гидростатика и балластные танки — data/demo/*.csv/json).

## Как запустить

1. Установите зависимости:

   ```
   pip install -r app/requirements.txt
   ```

2. Запустите приложение:
   ```
   uvicorn app.main:app --reload
   ```

3. Откройте [http://localhost:8000](http://localhost:8000) в браузере.

## Структура данных

- app/main.py — основной backend
- app/templates/ — все HTML/Jinja2 шаблоны
- app/static/ — стили/Bootstrap
- data/demo/ — demo-профиль судна с файлами гидростатики и танков

## Сценарий демонстрации

1. На главной выберите demo-профиль.
2. Введите осадки (по каждому борту и шпильке), плотность воды.
3. Введите sounding по каждому балластному танку.
4. Получите автоматический расчет (и детали по танкам).

## Для добавления своих судов

- Создайте json-профиль вида demo.json (см. demo).
- Добавьте свои таблицы гидростатики и танков (CSV).
- Загрузите их через форму.
